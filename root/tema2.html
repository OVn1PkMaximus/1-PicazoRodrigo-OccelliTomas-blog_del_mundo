Microblogging

El hecho de empezar una buena dieta requiere de una fuerza de voluntad por parte de la persona para dejar de comer algunas comidas
La dieta es algo muy importante, si no que de lo mas importtante. Una buena alimentacion es fundamental para poder seguir adelante con el ejercicio y tener buenos resultados
Tener las horas de sueño necesarias para poder tener resultados es algo muy importante.El dormir ayuda a los alimentos a generar buena sintesis de los nutrientes
El uso de suplementos puede ayudar en la alimentacion, pero no lo es todo. Los suplemento simplemente remplazan lo que puedes obtener de las comidas, solamente facilitan elpoder tener nutrientes mas rapido
Para que lña alimentacion sea optima se necesita beber mucha alguna, 2 a 3 litros de agua son necesarios para poder tener una buena hidratacion y poder tener un mejor rendimiento a la hora de hacer ejercicio.
